##  Hadits ke 398 dalam Riyadush Shalihin
- ![Screenshot 2026-02-27 at 05.35.44.png](/img/user/40%20-%20Obsidian/Attachment/Screenshot%202026-02-27%20at%2005.35.44.png)
 - Dari an-Nu'man radhiyallahuanhu, neraka paling ringan siksa nya diletakan bara api di telapak dua kaki dan otak (bagian atas tubuh) mendidih.

</div></div>

- Bagaimana **menakutkannya** adzab yang digambarkan sampai bara api yang menyentuh kulit bisa membuat bagian tubuh dalam dan di bagian atas mendidih. 
- Hal ini harus membuat kita **khawatir** akan pedihnya api neraka.
- Penggambaran tidak menyebutkan antara kulit punggung dengan telapak kaki, lutut, betis, paha, pusar, perut, ataupun leher. Tapi ini sampai pada otak, bagian dalam dan bagian atas.
- Dalam di Ramadhan ini ***Allah ﷻ*** memiliki hamba-hamba yang ***Allah ﷻ*** bebaskan dari api neraka, dan itu terjadi di setiap malam. Semoga kita termasuk hamba-hamba yang dibebaskan dari api neraka.
